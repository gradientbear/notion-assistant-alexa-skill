# Alexa Skill Sample Dialogs (Rich Test & User Guide)

Below are extensive sample dialogs covering **tasks**, **notes**,
**meals**, **workouts**, **energy**, and **shopping lists**.\
Each example shows realistic, natural Alexa interactions for testing and
user education.

------------------------------------------------------------------------

## üìù Task Management

### Add Tasks (Phrase-Based)

-   "Alexa, add **finish client report**"
-   "Alexa, add **buy milk and eggs** to my list"
-   "Alexa, add task **call mom tomorrow**"
-   "Alexa, please add **email the design team**"
-   "Alexa, add **prepare presentation slides**"

### Add Tasks (Structured)

-   "Alexa, add a **high** priority **work** task"
-   "Alexa, add a **work** task due **Monday**"
-   "Alexa, add a **high** priority task **finish client report**"
-   "Alexa, add a **personal** task for **Friday**"
-   "Alexa, add a **recurring** task every **weekly**"

### Complete Tasks

-   "Alexa, complete **finish report**"
-   "Alexa, mark **buy groceries** as done"
-   "Alexa, I finished **prepare presentation**"
-   "Alexa, complete task **call mom**"

### Delete Tasks

-   "Alexa, delete **buy milk**"
-   "Alexa, remove **email client**"
-   "Alexa, delete task **team meeting prep**"

### Update Status

-   "Alexa, set **finish report** to **in progress**"
-   "Alexa, move task **gym session** to **done**"
-   "Alexa, change status of **team meeting prep** to **to do**"

### Query Tasks

-   "Alexa, what are my tasks?"
-   "Alexa, read my **work** tasks"
-   "Alexa, what are my **high priority** tasks?"
-   "Alexa, what's due **tomorrow**?"
-   "Alexa, what did I complete this week?"
-   "Alexa, when is my next deadline?"
-   "Alexa, how many tasks do I have?"
